# Wireframes HTML - HAMMER Subastas

Esta carpeta contiene los wireframes de la aplicación HAMMER Subastas convertidos a HTML puro para facilitar su edición.

## Archivos Disponibles

1. **home.html** - Wireframe de la página de inicio
   - Header con logo y botones
   - Sección hero
   - Grid de categorías
   - Sección "Cómo funciona"
   - Footer

2. **login.html** - Wireframe de la página de login
   - Formulario de inicio de sesión
   - Campos de email y contraseña
   - Enlaces de "Olvidé mi contraseña" y "Registrarse"

3. **dashboard.html** - Wireframe del dashboard de usuario
   - Grid de estadísticas
   - Subastas activas
   - Feed de actividad reciente
   - Próximas subastas
   - Navegación inferior (mobile)

4. **auctions.html** - Wireframe del listado de subastas
   - Sidebar de filtros
   - Lista de subastas con imágenes
   - Toolbar con ordenamiento
   - Paginación

5. **live-auction.html** - Wireframe de subasta en vivo
   - Información del producto
   - Puja actual destacada
   - Formulario de puja con slider
   - Botones de puja rápida
   - Barra inferior sticky (mobile)

## Características

- **Diseño Responsive**: Los wireframes incluyen media queries para adaptarse a diferentes tamaños de pantalla
- **CSS Inline**: Todo el CSS está contenido en cada archivo para facilitar la edición
- **Estructura Semántica**: Se utilizan elementos HTML5 semánticos (header, nav, main, section, article)
- **Sin Dependencias**: Archivos HTML puros sin necesidad de frameworks o librerías

## Cómo Usar

1. Abre cualquier archivo `.html` en tu navegador web
2. Edita el HTML o CSS directamente en un editor de texto
3. Recarga el navegador para ver los cambios

## Paleta de Colores

Los wireframes utilizan una paleta monocromática (blanco y negro) para enfocarse en la estructura y el layout:

- **Negro (#000000)**: Elementos principales y bordes
- **Gris Oscuro (#1a1a1a)**: Fondos oscuros
- **Gris Medio (#999999)**: Elementos secundarios
- **Gris Claro (#cccccc)**: Fondos de contenido
- **Gris Muy Claro (#f3f4f6)**: Fondo de página

## Notas

- Estos wireframes son representaciones visuales de bajo fidelidad
- Los elementos son cajas sin contenido real para facilitar la visualización de la estructura
- Para ver la implementación real con colores y contenido, consulta los archivos `.tsx` en `/src/app/pages/`
